An element that shows a tooltip for another element... It's a tooltip.

Constructor

new CoreTooltipElement()

Creates a CoreTooltip element and attaches the shadow root.
Properties:
Name Type Description
placement String the side of the parent which the tooltip is shown
content String the content of the tooltip
openDelay Number the delay of appearance, in millisecond
closeDelay Number the delay of disappearance, in millisecond
for String the id of the target. If none specified, then it defaults to the parent
focusable Boolean whether to allow focus to show tooltip
manual Boolean whether mouseenter and mouseleave change the tooltip
noVisibleArrow Boolean whether to remove the arrow
