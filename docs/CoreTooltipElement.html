Class: CoreTooltipElement

CoreTooltipElement()

An element that shows a tooltip for another element... It's a tooltip.

Constructor

new CoreTooltipElement()

Creates a CoreTooltip element and attaches the shadow root.
Properties:
Name Type Description
placement String the side of the parent which the tooltip is shown. The attribute name is 'placement'.
content String the content of the tooltip. The attribute name is 'content'.
openDelay Number the delay of appearance, in millisecond. The attribute name is 'open-delay'.
closeDelay Number the delay of disappearance, in millisecond. The attribute name is 'close-delay'.
for String the id of the target. If none specified, then it defaults to the parent. The attribute name is 'for'.
focusable Boolean whether to allow focus to show tooltip. The attribute name is 'focusable'.
manual Boolean whether mouseenter and mouseleave change the tooltip. The attribute name is 'manual'.
noVisibleArrow Boolean whether to remove the arrow. The attribute name is 'no-visible-arrow'.
disabled Boolean whether it is disabled. The attribute name is 'disabled'.
offset Boolean The tooltip offset from the target, which is also dependent on placement position. It is in pixels. The attribute name is 'offset'.
