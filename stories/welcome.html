Brief Documentation

Here is our brief introduction on the list of our available web component and how to use them

Hello components

This component will generate a simple greetings

<core-hello></core-hello>

Below is the list of available attributes

  1. Language

    Change the language of the greeting

    <core-hello lang="en"></core-hello>

    Language component's value

    Accepted Attribute ValueDescription
    enEnglish language
    esSpanish language
    jpJapanese language
    frFrench language
  1. Rainbow coloring

    Add rainbow coloring

    <core-hello rainbow></core-hello>

Slider Components

This component will generate a slider component

<core-slider></core-slider>

Below is the list of available attributes

  1. min and max

    Change the minimum and maximum value of the slider

    <core-slider min='0' max='100'></core-slider>
  1. step

    Change the amount of each increment of the slider

    <core-slider step='10'></core-slider>
  1. disabled

    Change the status of the slider to disabled

    <core-slider disabled></core-slider>
  1. vertical

    Change the slider into vertical position

    <core-slider vertical></core-slider>
  1. rainbow

    Change the color of the slider bar into rainbow

    <core-slider rainbow></core-slider>

Tool tip Components

This component will generate a tool tip, an explanatory boxes that will appear on another element

<core-tooltip></core-tooltip>

Below is the list of available attributes

  1. content

    Change the content of the tool tip 

    <core-tooltip content='tooltip'></core-tooltip>
    <core-tooltip>tooltip</core-tooltip>

  1. open-delay

    Change the opening delay of the tool tip

    <core-tooltip open-delay='1000'></core-tooltip>
  1. close-delay

    Change the closing delay of the tool tip

    <core-tooltip close-delay='1000'></core-tooltip>
  1. focusable

    Change the status of the slider to disabled

    <core-tooltip focusable></core-tooltip>
  1. placement

    Change the slider into vertical position

    <core-tooltip placement='left'></core-tooltip>
  1. tabindex

    Change the tab index 

    <core-tooltip tabindex=0></core-tooltip>
  1. offset

    Change the offset of the tool tip above the component

    <core-tooltip offset=10></core-tooltip>
  1. no-visible-arrow

    Hide the arrow on the tool tip

    <core-tooltip no-visible-arrow></core-tooltip>
  1. effect

    Change the effect between dark of light

    <core-tooltip effect='dark'></core-tooltip>

    Effect value

    Accepted Attribute ValueDescription
    darkchange the theme into dark
    ligthchange the theme into light

  1. disabled
    <core-tooltip disabled></core-tooltip>

Link Components

This component will generate a link on your document

<core-link><core-link>

Below is the list of available attributes

  1. href

    Specifies the url of the link

    <core-link href="#"></core-link>
  1. type

    Change the type of the link

    <core-link type="danger"></core-link>

    Type Value

    Accepted Attribute ValueDescription
    primarylink with blue color
    successlink with lime color
    warninglink with orange color
    dangerlink with tomato color
    infolink with slate gray color
  1. underline

    Gives underline to the link

    <core-link underline></core-link>
  1. disabled

    Disables the link

    <core-link disabled></core-link>
